Prática de Debugging

Este repositório contém exercícios e práticas voltados para o aprendizado de técnicas de debugging (depuração) em códigos.

📚 Sobre o projeto

O foco deste projeto é ajudar a entender como identificar, analisar e corrigir erros em diferentes linguagens e ambientes de programação.

🚀 Tecnologias e ferramentas usadas

Linguagem usada: (JavaScript)

Ferramentas de depuração (ex: Chrome DevTools, VSCode Debugger)

🎯 Objetivos de aprendizado

Aprender a usar ferramentas de debugging

Entender como rastrear erros no código

Praticar a correção de bugs em cenários reais

Melhorar a qualidade do código com técnicas de depuração

▶️ Como usar este repositório

Explore os exemplos e exercícios em pastas específicas e teste as técnicas de debugging indicadas.

📌 Observação

Este projeto foi criado como prática de aprendizado e não tem finalidade comercial.
